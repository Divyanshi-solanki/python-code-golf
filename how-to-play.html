⛳ What is Code Golf?

Code Golf is the art of solving programming challenges using as few characters as possible. Like golf, the lowest score wins. You'll need to think creatively — using Python's built-ins, one-liners, and clever tricks to shrink your code to its smallest possible form.

🎮 Game Rules

  • Each challenge requires you to write a Python function named f() that returns the correct output.
  • Your solution is tested against multiple hidden test cases automatically.
  • All test cases must pass before you can submit.
  • Your score is based on character count — fewer characters = higher score.
  • Beat the "par score" for a bonus 200 points.
  • Solve in under 3 minutes for a speed bonus of 100 points.
  • Your personal best per challenge is saved automatically.

⌨️ Keyboard Shortcuts

  • Ctrl + Enter — Run your code
  • Tab — Insert 4 spaces

🐍 Python Golf Tips & Tricks

List Comprehensions
# Long way (4 lines)
result = []
for x in lst:
    if x > 0:
        result.append(x*2)

# Golf way (1 line)
[x*2 for x in lst if x>0]
Lambda + Built-ins
# Long way
def f(lst):
    return sum(lst)/len(lst)

# Golf way
f=lambda l:sum(l)/len(l)
Ternary Operators
# Long way
if x > 0:
    return "pos"
else:
    return "neg"

# Golf way
return"pos"if x>0 else"neg"

String Slicing
s[::-1]     # reverse
s[::2]      # every 2nd char
s[1:-1]     # strip first+last

reversed(s)  # longer!
